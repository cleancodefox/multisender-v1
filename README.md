### Prerequisites

### Installation

```bash
# Clone the repository
git clone <YOUR_GIT_URL>
cd multisent

# Install dependencies
npm install

# Start development server
npm run dev
```

The application will be available at `http://localhost:5173`

### Building for Production

```bash
# Build the project
npm run build

# Preview the build
npm run preview
```
